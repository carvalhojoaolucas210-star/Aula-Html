Chart.js — Exemplos Práticos

7 exemplos funcionais usando dados de chamados de suporte. Cada gráfico aplica as técnicas descritas no material didático.

1

Barras Verticais — type: "bar"

Gráfico mais básico. Compara chamados por tipo (Bug vs Melhoria). Cada barra recebe uma cor via backgroundColor como array e os cantos são arredondados com borderRadius: 8.

2

Barras Empilhadas — stacked: true + datalabels

Bug vs Melhoria empilhados por mês. Exige scales.x.stacked: true e scales.y.stacked: true. O plugin datalabels exibe os valores sobre cada segmento, ocultando zeros.

Bug Melhoria
3

Doughnut (Rosca) — type: "doughnut" + legenda customizada

Distribuição por categoria. cutout: "52%" controla o buraco central. A legenda usa generateLabels para exibir a porcentagem. datalabels só aparece em fatias > 8%.

4

Gráfico de Linha — type: "line" + fill + tension

Evolução das horas gastas por mês. fill: true preenche a área abaixo da linha. tension: 0.35 suaviza as curvas. pointBackgroundColor: "#fff" cria o efeito de bolinha vazada.

5

Barras Horizontais — indexAxis: "y"

Tempo médio por categoria. A única mudança é indexAxis: "y" — uma propriedade que transforma barras verticais em horizontais, invertendo os papéis dos eixos X e Y.

6

Múltiplas Linhas — 2 datasets tipo line

Duas linhas no mesmo gráfico: Bug (vermelho) e Melhoria (indigo). A função datalabels.color lê ctx.dataset.borderColor para que o rótulo combine com a linha correspondente.

Bug Melhoria
7

Destruir e Recriar — chart.destroy() + referência global

O mesmo <canvas> reutilizado para 3 tipos de gráfico. É obrigatório chamar chart.destroy() antes de criar um novo no mesmo canvas. A referência fica em chartEx7.
